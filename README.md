#タイピング記録アプリ

Javascriptの学習及びポートフォリオように作成した、タイピングの練習記録を保存できるWebアプリです。

##アプリのURL
https://chiharu0175.github.io/typing_record/

##主な機能
- タイピングをした日にちを選択
- そのときの結果の画像をアップロード
- 最後にコメントを付けて次の練習に活かします。
- 入力した記録はブラウザ(`localStorage`)に保存され、ページを閉じても消えない。
- 不要になった記録は「削除」ボタンで個別に消すことができる。

使った技術：
- HTML / CSS
- JavaScript

工夫した点：
・段落同士の隙間を設けて、見やすいサイト作りを心がけた。
・デザインの4原則の「繰り返し」に基づき、段落の下にある下線の色はすべて同じにした。
・ユーザがデータを入力するところをわかりやすくするために背景色を設けた。タイトルの背景色を入力欄と同一にすることで統一感を出した。
・文字を中央揃えにすることで見やすいサイト作りを心がけた。
・`localStorage` を使ったデータの保存と読み込みの仕組みを実装した。
・Git / GitHub を使ったバージョン管理とデプロイの基礎を学んだ。
